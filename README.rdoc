=== FAQS app

An app/site template for serving up content such as FAQs or other post-ishy type things. Columnar output on the page, sorted by category. ActiveAdmin on the back for a sweet CMS feel. 

- Devise
- ActiveAdmin
- Postgres
- Bootstrap
